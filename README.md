# Airport Transfers - Premium Executive Service

Este proyecto es una plataforma web para la reserva de traslados ejecutivos y VIP desde y hacia el aeropuerto, con un enfoque en la elegancia, seguridad y una experiencia de usuario premium.

## 🤖 Uso de Inteligencia Artificial en el Desarrollo

Durante el desarrollo de este proyecto, utilizamos un **Asistente de Codificación de IA (Gemini/Antigravity)** para acelerar y mejorar el proceso de construcción de la plataforma. 

**¿Cómo nos ayudó la IA?**
- **Diseño y Maquetación UI/UX:** Asistió en la creación de una interfaz moderna, atractiva e interactiva utilizando Tailwind CSS, incluyendo la implementación de estilos "glassmorphism" y modo oscuro.
- **Estructuración y Refactorización:** Nos ayudó a extraer los estilos en línea hacia un archivo `styles.css` externo para tener un código más limpio y fácil de mantener.
- **Correcciones de Accesibilidad (A11y):** Identificó y resolvió problemas de accesibilidad (por ejemplo, el error "Links must have discernible text" al agregar las etiquetas `aria-label` y `title` a nuestros iconos interactivos).
- **Funcionalidad del Formulario:** Apoyó en el diseño de la lógica en JavaScript para el formulario de reservas de múltiples pasos, el cálculo de las tarifas estimadas y la integración directa con la API de WhatsApp.
